# 🔐 RBAC Access Control System

## 📌 Project Overview
This project implements a **Role-Based Access Control (RBAC)** system where users are assigned roles, and roles define permissions to access system resources.

---

## 🚀 Features
- 🔑 JWT Authentication
- 👥 User Management
- 🛡 Role Management
- 🔐 Permission Management
- 📊 Access Logs Tracking
- 🔒 Secure API access

---

## 🛠 Tech Stack
- **Backend:** Spring Boot
- **Frontend:** Angular
- **Database:** MySQL
- **Authentication:** JWT

---

## 🧩 Architecture
User → Role → Permission → Resource → Access Logs
